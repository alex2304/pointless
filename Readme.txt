Визуализация классической школьной игры "Точки" (The Pointless Game)

Описание:
В игру играют два человека. Имеется клетчатое поле, на котором игрокам предлагается расставлять точки определённых цветов (цвета игроки выбирают себе перед началом игры). Соседними являются точки, расстояние между которыми равно стороне клетки (по вертикали, горизонтали или диагонали). Если соседние точки, расставленные одним игроком, образуют замкнутую геометрическую фигуру, то игрок может соединить их. Все точки другого игрока, оказавшиеся в этой области, считаются "завоёванными". Цель игры - "поймать" как можно больше точек соперника. Игра заканчивается, когда один из игроков поднимает белый флаг, либо же по истечению свободных клеток на поле (поле имеет ограниченный размер).

Правила и уточнения:
1. Игроки ходят по очереди. Каждый игрок может ставить точку в любое свободное место на поле, за исключением области, которая уже была обведена. После захвата области игрок имеет дополнительный ход.
2. Захватывать можно только те области, в которых есть хотя бы одна точка соперника. Все точки захваченной области становятся недействительными, то есть их нельзя использовать для создания новых областей. Ставить точки в уже захваченные области не имеет смысла. Если вам больше некуда ставить точки, то ваша жизнь - тлен, поднимайте белый флаг и прыгайте за борт. Карамба:)
3. Цель игры - захватить как можно больше точек соперника. По окончании игры побеждает игрок, захвативший больше точек.
